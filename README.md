# Homebrew for okta-cli

This is the homebrew-tap for [okta-cli](https://github.com/flypenguin/okta-cli),
a 3rd party, unofficial okta CLI wrapper.

## Installation

```bash
brew tap flypenguin/okta-cli
brew install okta-cli
```
